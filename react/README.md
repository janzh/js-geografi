# Triage-frontend

## Local development

### Install dependencies
`npm install`

### Run application
`npm run start`

### Run tests
`npm run test`

#### Check test coverage in the application
`npm run test:coverage`

### Check formatting and linting, tests
`npm run app:check`

### Fix formatting and linting errors
`npm run app:fix`

### Build application
`npm run build`

### Test docker file
`docker run -p 8080:8080 $(docker build -q .)`
